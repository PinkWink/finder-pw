PWFinder — 4분할 파일 탐색기 작업 히스토리

우분투 기본 탐색기를 대체할, 화면을 최대 4분할로 나누고 각 패널에 색상·이름·git 상태·터미널·트리 사이드바를 둔 데스크톱 탐색기. Tauri 2 + React 18 기반으로 처음부터 빌드, GitHub 게시, APT 자동 배포까지 완료.

개요

사용자(PinkWink)는 우분투 기본 파일 매니저(Nautilus)의 화면 분할 미지원이라는 한계를 해결하기 위해, 최대 4개의 분할창을 가지고 각 분할창에 색상·이름을 부여하고 git 상태·터미널 단축까지 통합된 탐색기 도구를 만들고자 했다. 단일 세션 내에서 Rust·Tauri·React 셋업 → 기능 구현 → GitHub 게시 → 자체 APT 배포 → 두 차례 릴리스(v0.1.0 → v0.1.1 → v0.1.2) 까지 진행됐다.

의존성 자동 설치되는가?
맥은 일단 나중에 하고, 근데, 설치할때 dependency들이 모두 따라 설치되는걸까? 유저들이 설치 해야할까?
